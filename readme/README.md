# Food Recipe Database

## Technologies Used
- PHP
- MySQL
- HTML, CSS
- JavaScript (Ajax)

## Features
- Add, View, Edit, Delete recipes (CRUD)
- Upload recipe images
- Ingredients stored as JSON
- Search recipes
- Ajax ingredient autocomplete
- Secure coding using PDO and output escaping

## Setup Instructions
1. Import database.sql into phpMyAdmin
2. Update config/db.php credentials
3. Upload project to server
4. Open public/index.php

## Security
- SQL Injection prevention using prepared statements
- XSS prevention using htmlspecialchars()
